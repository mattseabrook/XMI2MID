# Build

Use the Command Prompt for Visual Studio 2022 to run `build.cmd` included in this repository.

```cmd
build.cmd
```

# Usage

Supply the path to the XMI file as the first argument.

```cmd
xmi2mid.exe "C:\Path\To\XMI\File.xmi"
```
